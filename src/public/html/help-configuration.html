Help
  • Open Video: Ctrl + O
  • Open Image Directory: Ctrl + I
  • Open TFRecord Directory: Ctrl + Space
  • Save Tags: Ctrl + S
  • Toggle Play/Pause: Space
  • Toggle Tracking: Ctrl + T
  • Toggle Exclusive Add Mode: Crtl + N
  • Export Tags: Ctrl + E
  • Active Learning: Ctrl + L
  • Duplicate last frame regions: D
  • Delete Frame: Shift + Del
  • Open Developer Console: Ctrl + D
  • Refresh App: Ctrl + R
  • Show Help: Ctrl + H
  • Rect box creation: press R key.
    • Two-point mode: Hold down Ctrl while creating a region
    • Square mode: Hold down Shift while creating a region
  • Template based box creation: press T key.
  • Example based box creation: press Y key. Uses currently selected region as a template for size.
  • Exclusive selection: press L key to block/unblock regions manipulation.
  • Multi-select: Hold down Shift while selecting regions
  • Select next: Tab
  • Select All: Ctrl + (A or 1 on number pad)
  • Cut: Ctrl + X
  • Copy: Ctrl + C
  • Paste: Ctrl + V

VOTT allows you to fine tune the bounding boxes using the arrow keys in a few different ways. While a region is selected:

  • Move region: Hold Ctrl + Direction
  • Shrink region:: Hold Ctrl + Alt + Direction
  • Expand region: Hold Ctrl + Shift + Direction

Tuning visual display of regions:

  • Toggle region background: Ctrl + B

Use the number keys to quickly tag a selected region
(Only works for single digits 0-9)

To delete tags from list, use ctrl+left click on the tag you want removed. It will remove the tag from the list and untag any images that use that tag.
